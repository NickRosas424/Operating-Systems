One operating system function is starting up the computer. After pressing the power button a series of start up tests run through the computer's system.
After the tests are successful the kernel of the operating system load into its memory.
Another operating system function is providing a user interface. This controls how users manage their computers. Menus, windows, and instructions are 
displayed on the screen.
A third operating system function is managing programs. Users are able to multitask by having several applications open in different windows. The application
being used appears in the foreground while the others run in the background.
